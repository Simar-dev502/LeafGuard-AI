# 🌿 LeafGuard AI — Plant Disease Detection System

An AI-powered web application that detects plant diseases from leaf images using deep learning and provides smart insights for farmers and researchers.

---

## 🚀 Features

* 🌱 Upload plant leaf image
* 🤖 AI-based disease detection (CNN model)
* 📊 Confidence score display
* 🧠 Smart disease description (AI / rule-based)
* 🛡️ Prevention suggestions
* 🎨 Modern responsive UI with animations
* 🔊 Text-to-Speech support

---

## 🧠 Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Flask (Python)
* **AI Model:** TensorFlow / Keras (Transfer Learning - MobileNet/ResNet)
* **Image Processing:** PIL, NumPy
* **Other:** gTTS, Requests

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/Simar-dev502/LeafGuard-AI.git
cd LeafGuard-AI
```

### 2️⃣ Create virtual environment

```bash
python -m venv tfenv
source tfenv/bin/activate   # Windows: tfenv\Scripts\activate
```

### 3️⃣ Install dependencies

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the app

```bash
python backend/app.py
```

👉 Open in browser:

```
http://127.0.0.1:5000
```
